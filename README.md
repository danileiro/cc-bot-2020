# cc-bot-2020
Projeto Integrador 2º Semestre 2020


### Execução

Ambiente virtual python 3.6 incluso no repositório.


```sh
$ cd cc-bot-2020
$ source ./venv_bot/bin/activate
$ python bot.py
```
###### (OBS) Por movitos de segurança a chave de API do Bot não está inclusa!
Para adicionar nova chave de API no código
arquivo bot.py
```python
TOKEN = "API-KEY-HERE"
```
